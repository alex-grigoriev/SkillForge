# Реализация умного указателя (`shared_ptr_toy`)

## 📌 Описание

В этом задании необходимо реализовать упрощённый аналог умного указателя `shared_ptr` — класс **`shared_ptr_toy`**, который управляет временем жизни объектов класса `Toy`.

Основная идея: несколько объектов `shared_ptr_toy` могут владеть одной игрушкой (`Toy`), а удаление происходит автоматически, когда последняя ссылка исчезает.

---

## 🎯 Цель

Реализовать:

* класс `shared_ptr_toy`
* механизм **подсчёта ссылок (reference counting)**
* автоматическое освобождение памяти
* функцию `make_shared_toy`

---

## 🧠 Принцип работы

Каждый `shared_ptr_toy` содержит:

```cpp
Toy* ptr;     // указатель на объект
int* count;   // указатель на счётчик ссылок
```

### Как это работает:

* При создании → `count = 1`
* При копировании → `count++`
* При уничтожении → `count--`
* Если `count == 0` → удаляются:

  * объект `Toy`
  * сам счётчик

---

## ⚙️ Требования к классу

### 🔹 Конструкторы

1. **По умолчанию**

```cpp
shared_ptr_toy();
```

* создаёт пустой указатель (`nullptr`)

---

2. **С параметром (имя игрушки)**

```cpp
shared_ptr_toy(const std::string& name);
```

* создаёт новый `Toy`
* создаёт счётчик = 1

---

3. **Копирующий конструктор**

```cpp
shared_ptr_toy(const shared_ptr_toy& other);
```

* копирует указатели
* увеличивает счётчик

---

### 🔹 Оператор присваивания

```cpp
shared_ptr_toy& operator=(const shared_ptr_toy& other);
```

Алгоритм:

1. Проверка на самоприсваивание
2. Освобождение текущего ресурса
3. Копирование указателей
4. Увеличение счётчика

---

### 🔹 Деструктор

```cpp
~shared_ptr_toy();
```

* уменьшает счётчик
* удаляет `Toy`, если это последняя ссылка

---

## 🧩 Обязательные методы

### 🔸 reset()

```cpp
void reset();
```

* освобождает текущий объект
* делает указатели `nullptr`

---

### 🔸 get()

```cpp
Toy* get() const;
```

* возвращает сырой указатель

---

### 🔸 use_count()

```cpp
int use_count() const;
```

* возвращает количество ссылок

---

### 🔸 getToyName()

```cpp
std::string getToyName();
```

* возвращает имя игрушки
* если `nullptr` → `"Nothing"`

---

## 🏭 Функция-фабрика

```cpp
shared_ptr_toy make_shared_toy(const std::string& name);
```

* создаёт и возвращает `shared_ptr_toy`
* аналог `std::make_shared`

---

## ▶️ Пример использования

```cpp
shared_ptr_toy toy_01 = make_shared_toy("ball");
shared_ptr_toy toy_02(toy_01);
shared_ptr_toy toy_03("duck");
```
