# 📞 PhoneBook (C++)

Простое консольное приложение на **C++**, реализующее телефонную книгу с возможностью добавления и поиска записей.

---

## 🚀 Возможности

- ➕ Добавление записи (номер + фамилия)
- 🔍 Поиск по номеру телефона
- 🔎 Поиск по фамилии
- ⚠️ Обработка ошибок ввода
- 🧠 Нормализация телефонного номера

---

## 🏗️ Архитектура

Минималистичная реализация в одном файле:

main.cpp → логика приложения и обработка команд

---

## 📦 Используемые технологии

- **C++ (STL)**
- `std::map` — хранение телефонной книги
- `std::string` — работа со строками

---

## 📌 Формат номера

12-345678


Внутри номер преобразуется в числовой ключ (удаляются разделители).

---

## ▶️ Запуск

### Сборка

```bash
g++ main.cpp -o phonebook
```
Запуск

```
./phonebook
```

💻 Использование

Доступные команды:

➕ Добавление
add

Далее:

Insert phone number: 12-345678
Insert Surname: Ivanov
🔍 Поиск по номеру
phone
🔎 Поиск по фамилии
sname
❌ Выход
q

🧠 Особенности реализации
Используется std::map<int, std::string>:
ключ — номер телефона
значение — фамилия
Номер преобразуется в число через stoi
Поиск по фамилии выполняется полным перебором (O(n))

⚠️ Ограничения
Нет поддержки:
имён (только фамилия)
нескольких номеров на человека
Формат номера жёстко задан
Используется int → возможны ограничения по размеру номера
Нет сохранения данных (всё в памяти)
