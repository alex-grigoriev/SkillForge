# Задание 9. Пупырка

## 📌 Описание

В этом задании требуется реализовать консольную программу, имитирующую процесс лопания пузырчатой упаковочной плёнки (пупырки).

Плёнка представляет собой поле размером **12 × 12**, где каждый элемент — это пузырёк, который может находиться в одном из двух состояний:

* `true` — пузырёк целый
* `false` — пузырёк лопнут

---

## 🎯 Цель

Реализовать программу, в которой пользователь:

* видит текущее состояние пупырки
* задаёт прямоугольный регион
* «лопает» все пузырьки внутри него

Программа завершается, когда **все пузырьки будут лопнуты**.

---

## 🧠 Принцип работы

1. Создаётся двумерный массив:

```cpp
bool bubble[12][12];
```

2. Инициализация:

* все элементы устанавливаются в `true`

3. Пользователь вводит координаты:

```
x1 y1 x2 y2
```

4. Программа:

* проверяет корректность координат
* определяет прямоугольную область
* лопает пузырьки внутри неё

---

## 📥 Входные данные

На каждом шаге пользователь вводит:

```
x1 y1 x2 y2
```

где:

* `(x1, y1)` — первая точка региона
* `(x2, y2)` — противоположная точка

---

## 📤 Выходные данные

1. Отображение текущего состояния:

```
o — целый пузырёк
x — лопнутый
```

2. При лопании каждого нового пузырька:

```
Pop!
```

3. При завершении:

```
All bubbles popped!
```

---

## ⚙️ Требования

* Использовать **двумерный массив типа bool**
* Реализовать отдельные функции:

  * инициализация
  * вывод
  * лопание региона
  * проверка окончания
* Проверять корректность координат
* Лопать только выбранный регион (не всё поле)

---

## 🧩 Рекомендуемые функции

### 🔹 Инициализация

```cpp
void init(bool bubble[12][12]);
```

---

### 🔹 Отображение

```cpp
void print(bool bubble[12][12]);
```

---

### 🔹 Проверка координат

```cpp
bool isValid(int x, int y);
```

---

### 🔹 Лопание региона

```cpp
void popRegion(bool bubble[12][12], int x1, int y1, int x2, int y2);
```

---

### 🔹 Проверка завершения

```cpp
bool hasBubbles(bool bubble[12][12]);
```

---

## ⚠️ Важные детали

* Координаты могут быть введены в любом порядке → требуется нормализация
* Лопать только **целые** пузырьки
* Не выводить `Pop!`, если пузырёк уже лопнут
* Проверять границы массива (0–11)

---

## 🚫 Частые ошибки

* ❌ отсутствие проверки границ
* ❌ повторное лопание уже лопнутых пузырей
* ❌ обработка всей матрицы вместо региона
* ❌ неправильная обработка координат (x1 > x2)
* ❌ программа не завершается
