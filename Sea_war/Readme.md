# Морской бой

## 📌 Описание

Необходимо реализовать консольную версию упрощённой игры **«Морской бой»** для двух игроков.

Игровое поле каждого игрока — **10 × 10 клеток**.

---

## ⚓ Корабли

Каждый игрок имеет одинаковый набор:

| Тип корабля | Размер | Количество |
| ----------- | ------ | ---------- |
| Маленький   | 1      | 4          |
| Небольшой   | 2      | 3          |
| Средний     | 3      | 2          |
| Большой     | 4      | 1          |

---

## 📥 Ввод данных

### Размещение кораблей

* **Размер 1** → вводится одна точка:

  ```
  x y
  ```

* **Размер ≥ 2** → вводятся начало и конец:

  ```
  x1 y1 x2 y2
  ```

---

## ⚠️ Ограничения

* Только **горизонтальное или вертикальное** размещение
* ❌ Диагональ запрещена
* ❌ Выход за границы поля запрещён
* ❌ Наложение кораблей запрещено
* ✔️ Корабли могут соприкасаться

---

## 🎮 Игровой процесс

1. Игроки по очереди расставляют корабли
2. Затем начинается бой:

   * Игрок вводит координаты выстрела
   * Программа сообщает:

     * `Hit` (попал)
     * `Miss` (мимо)
3. При попадании клетка уничтожается
4. Игра заканчивается, когда все корабли одного игрока уничтожены

---

## 🏁 Победа

Побеждает игрок, уничтоживший все корабли противника.

---

* Скрытие поля противника
* Отображение попаданий (`X`) и промахов (`.`)
* Игра против бота
* Проверка «убит/ранен»
