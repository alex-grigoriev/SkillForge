# Механическое пианино

## 📌 Описание

В этом задании необходимо реализовать упрощённую модель механического пианино с использованием битовых масок.

Пианино имеет **7 клавиш (нот)**:

* DO, RE, MI, FA, SOL, LA, SI

Каждая нота кодируется **уникальным битом** в числе типа `int`.

---

## 🎯 Цель

* Принять от пользователя строку с цифрами от **1 до 7**
* Преобразовать её в **битовую маску**
* Вывести ноты, входящие в «аккорд»

---

## 🎵 Ноты и их битовые значения

```cpp
enum note
{
    DO  = 1,    // 1 << 0
    RE  = 2,    // 1 << 1
    MI  = 4,    // 1 << 2
    FA  = 8,    // 1 << 3
    SOL = 16,   // 1 << 4
    LA  = 32,   // 1 << 5
    SI  = 64    // 1 << 6
};
```

---

## 🧠 Принцип работы

### 1. Ввод

```text
1234
```

### 2. Преобразование в биты

Каждая цифра превращается в бит:

```cpp
mask |= (1 << (digit - 1));
```

👉 Например:

* `1 → DO → 0000001`
* `2 → RE → 0000010`
* `3 → MI → 0000100`

Итог:

```text
123 → 0000111
```

---

## 📥 Входные данные

Строка:

```text
1234
```

## 📤 Выходные данные

```text
DO RE MI FA
```

## ⚙️ Требования

* Использовать `enum` для нот
* Использовать битовые операции (`|`, `&`, `<<`)
* Игнорировать повторяющиеся ноты
* Обрабатывать строку посимвольно

---

## 🔍 Пример

### Ввод:

```text
1137
```

### Обработка:

* 1 → DO
* 1 → (игнорируется повтор)
* 3 → MI
* 7 → SI

### Вывод:

```text
DO MI SI
```

---

## ⚠️ Важные детали

* Повторные цифры не дублируются (бит уже установлен)
* Используется операция:

```cpp
notes |= ...
```

* Проверка:

```cpp
if (notes & NOTE)
```
