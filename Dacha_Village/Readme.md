# 🏡 Модель дачного посёлка (C++)

## 📌 Описание проекта

Данный проект реализует **модель дачного посёлка** с использованием структур данных в C++.
Основная цель — построить **гибкую, расширяемую и логически связанную систему**, описывающую участки, здания и помещения.

Программа позволяет пользователю:

* задать количество участков;
* заполнить информацию о каждом участке;
* указать здания и их параметры;
* описать структуру жилых домов (этажи, комнаты);
* рассчитать **процент застройки посёлка**.

---

## 🧠 Структура модели

Проект реализует иерархию объектов:

```text
Village (посёлок)
 └── Plot (участок)
      ├── Building (здания)
      │     ├── House (дом)
      │     │     └── Floor (этаж)
      │     │           └── Room (комната)
      │     ├── Bathhouse (баня)
      │     ├── Garage (гараж)
      │     └── Shed (сарай)
```

---

## 🧩 Используемые структуры

### 🏠 Комната

* Тип комнаты (спальня, кухня, ванная и т.д.)
* Площадь

### 🏢 Этаж

* Высота потолка
* Список комнат

### 🏗 Здание

* Тип здания:

  * дом
  * гараж
  * сарай
  * баня
* Площадь
* Наличие печи (для дома и бани)
* Список этажей (только для дома)

### 📍 Участок

* Номер
* Общая площадь
* Список построек

### 🌍 Посёлок

* Список участков

---

## ⚙️ Функциональность

* 📥 Ввод данных с клавиатуры
* 🧱 Создание вложенных структур (вектор векторе)
* 🔄 Универсальный механизм добавления объектов (`push_back`)
* 📊 Расчёт процента застройки:

```text
(суммарная площадь зданий / общая площадь участков) * 100%
```

---

## ⏱ Ввод данных

Пользователь последовательно вводит:

1. Количество участков
2. Для каждого участка:

   * площадь участка
   * количество зданий
3. Для каждого здания:

   * тип
   * площадь
   * наличие печи (если требуется)
4. Для дома:

   * количество этажей (1–3)
5. Для каждого этажа:

   * высота потолка
   * количество комнат (2–4)
6. Для каждой комнаты:

   * тип
   * площадь

---

## 📤 Вывод

После ввода всех данных программа выводит:

* 📊 Процент застройки посёлка

Пример:

```text
Total building coverage: 37.5%
```
