# 🎬 VideoPlayer

Desktop-приложение на **C++17 + Qt (Widgets + Multimedia)** для воспроизведения видеофайлов.

---

## 🚀 Возможности

- 📂 Открытие видеофайлов (`.mp4`, `.avi`)
- ▶️ Воспроизведение / ⏸ Пауза
- 🎚 Перемотка через слайдер
- 🔊 Аудио через `QAudioOutput`
- 🖥 Отображение видео (`QVideoWidget`)

---

## 🖼️ Интерфейс

- Видео-плеер (основная область)
- Слайдер прогресса
- Кнопки управления:
  - Open
  - Play
  - Pause

---

## 🏗️ Архитектура

- `main.cpp` — UI и логика плеера  
- `media.qrc` — ресурсы (если используются)

> В проекте также есть заготовка `QMainWindow`, но в текущей версии она не используется.

---

## ⚙️ Технологии

- **C++17**
- **Qt 6**
  - Widgets
  - Multimedia
  - MultimediaWidgets
- **CMake**

---

## 📦 Зависимости

Необходимо установить Qt со следующими модулями:

- Qt6::Widgets
- Qt6::Multimedia
- Qt6::MultimediaWidgets

---

### 🔧 Сборка (Windows / MinGW)

```bash
cmake -S . -B build
cmake --build build
```

Если Qt не найден:

```bash
cmake -S . -B build -DCMAKE_PREFIX_PATH="C:/Qt/6.x.x/mingw_64"
```

---

## ▶️ Запуск

```bash
cd build
./VideoPlayer.exe
```

## 📌 Использование

1. Нажми **Open**
2. Выбери видеофайл
3. Нажми **Play**
4. Перематывай через слайдер
5. Нажми **Pause** для паузы

## 🧠 Особенности реализации

- Используется `QMediaPlayer` для воспроизведения  
- `QVideoWidget` для вывода видео  
- Синхронизация позиции:
  - `durationChanged`
  - `positionChanged`
- Защита от “дёрганья” слайдера (buffer/balance логика)
